# Distribuciones Linux

<br />
Distribuciones Linux que se pueden utilizar para pruebas de pentesting, análisis forense, respuesta a incidentes, ingeniería inversa, threat hunting, análisis de malware, privacidad, anonimato y recolección, procesamiento y análisis de datos.  
<br />

Nombre | OVA | Especialidad | Distro Base | Descripción | Notas 
---- | ---- | ---- | ---- | ---- | ---- 
[Sans Investigative Forensics Toolkit (SIFT)](https://www.sans.org/tools/sift-workstation/) |Si|Incident Response, Forensic|Ubuntu|Es una colección de herramientas forenses y de respuesta a incidentes de código abierto y gratuitas|Requiere registro en SANS para descargar
[Kali Linux](https://www.kali.org/) |Si|Pentesting, Incident Response, Forensic|Debian|Orientada a diversas tareas de seguridad de la información, como pruebas de penetración, investigación de seguridad, informática forense e ingeniería inversa.|Permite varios dispositivos
[Parrot Security OS](https://www.parrotsec.org/) |Si|Pentesting, Incident Response, Forensic|Debian|Distribucion diseñada pensando en la seguridad y la privacidad. Pentesting, análisis forense digital e ingeniería inversa, desarrollo y privacidad.|Tiene 3 distribuciones
[BackBox ](https://www.backbox.org/) |No|Pentesting|Ubuntu|Análisis de aplicaciones web, análisis de redes, pruebas de estrés, rastreo, evaluación de vulnerabilidades, análisis forense informático, automoción y explotación.|
[BlackArch](https://blackarch.org/) |Si|Pentesting|Arch Linux|Distribución orientada a pruebas de penetración|
[Pentoo](https://www.pentoo.ch/) |No|Pentesting|Gentoo|Live distro diseñado para pruebas de penetración y evaluación de seguridad.|
[Samurai-WTF](https://github.com/SamuraiWTF/samuraiwtf) |No|Pentesting|Ubuntu|Samurai Web Testing Framework diseñado para pruebas de penetración y evaluación de seguridad.|Cambio de nombre a Samurai Web Training Framework
[samuraiwtf](https://github.com/SamuraiWTF/samuraiwtf) |Si|Training|Ubuntu|Samurai Web Training Framework es un framework diseñado para configurar rápidamente máquinas virtuales de entrenamiento con herramientas y objetivos de aplicaciones vulnerables.|Era el proyecto Samurai-WTF
[Network Security Toolkit (NST)](https://sourceforge.net/projects/nst/) |No|Security, Monitoring|Fedora|Es un Live CD con herramientas de análisis de red|
[Wifislax ](https://www.wifislax.com/) |No|WIFI Pentesting|Slackware|Orientada a pentesting de redes inalámbricas|
[Bugtraq](https://archiveos.org/bugtraq/) |No|Pentesting, forensic|Debian|Orientado a análisis forense digital, pruebas de penetración, laboratorios de malware y análisis forense GSM. |Sin soporte
[Cyborghawk](https://sourceforge.net/projects/cyborghawk1/) |No|Pentesting|Ubuntu|Distribución orientada a pruebas de penetración, evaluación de red, análisis forense digital, pruebas de seguridad móvil e inalámbrica.|
[kodachi](https://www.digi77.com/linux-kodachi/) |No|Anonimato|Debian|Live distro orientada al anonimato en la red|
[Whonix](https://www.whonix.org/) |Si|Anonimato|Debian|Consiste en 2 partes: Whonix-Gateway actúa como una compuerta con Tor, Whonix-Workstation está en una red completamente aislada. Permite usar aplicaciones y correr servidores anónimamente sobre la red. |
[Appliance for Digital Investigation and Analysis (AIDA) - LiFTeR](https://forensics.cert.org/) |Si|Forensic|Centos|Diseñada para el análisis forense, utiliza el repositorio LiFTeR  (The CERT Linux Forensics Tools Repository) de la universidad de Carnegie Mellon.|
[Computer Aided Investigative Environment (CAINE)](https://www.caine-live.net/) |No|Forensic|Ubuntu|Distro live especializada en análisis forense|
[SKADI](https://github.com/orlikoski/Skadi) |Si|Collect, Process, And Hunt|Ubuntu|Distro con herramientas que permiten la recolección, procesamiento y análisis avanzado de artefactos e imágenes forenses. |
[Tsurugi](https://tsurugi-linux.org/) |Si|Forensic|Ubuntu|Live distro especializada en análisis forense|
[RedHunt OS](https://github.com/redhuntlabs/RedHunt-OS) |Si|Threat Hunting|Lubuntu|Máquina virtual para emulación de adversarios y búsqueda de amenazas.|
[REMnux](https://remnux.org/ ) |Si|Threat Hunting|Ubuntu|Conjunto de herramientas de Linux para realizar ingeniería inversa y analizar malware|
[Web Security Dojo](https://www.mavensecurity.com/resources/web-security-dojo ) |si|Training|Ubuntu|Máquina virtual que proporciona las herramientas, los objetivos y la documentación para aprender y practicar las pruebas de seguridad de las aplicaciones web. |
[Metasploitable](https://github.com/rapid7/metasploitable3 ) |Si|Training|Ubuntu/Windows|Maquina virtual vulnerable con el cual realizar pruebas de seguridad, Hay 3 versiones: 1 y 2 para vulnerabilidades Linux y la 3 para Windows|
[OWASP Broken Web Apps](https://sourceforge.net/projects/owaspbwa/ ) |No|Training|Ubuntu|Entorno de pruebas de vulnerabilidades de aplicaciones web|
[Active Defense Harbinger Distribution (ADHD)](https://www.blackhillsinfosec.com/projects/adhd/ ) |Si|Pentesting|Ubuntu|Es una herramienta de monitoreo y defensa activa ante ataques|
[Slingshot](https://www.sans.org/tools/slingshot/ ) |Si|Pentesting|Ubuntu|Herramienta para el curso de SANS de pentesting.|Requiere registro en SANS para descargar
[Puppy](https://puppylinux.com/) |No|Desktop|Ubuntu|Live distro para ejecutar un entorno de escritorio linux|
